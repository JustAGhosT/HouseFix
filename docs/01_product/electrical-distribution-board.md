# Electrical Distribution Board — Replacement Plan

## Context

The house has an existing distribution board that needs replacing with a modern DIN rail board. This is a straight swap — replacing the old board with a new PVC flush-mount DIN rail 8-way board, re-terminating existing circuits onto new DIN rail breakers.

**Important:** All electrical work must be done by a qualified electrician. A Certificate of Compliance (CoC) is required after installation per SANS 10142-1.

---

## Board Specification

- **Board:** Distribution Board PVC Flush DIN Rail 8 Way (SKU 301657)
- **Mount:** Flush mount
- **Material:** Polypropylene (white casing, clear lid)
- **Dimensions:** 220mm × 210mm × 90mm
- **DIN rail:** Standard 35mm DIN rail

## Installation Order

1. Electrician isolates mains supply at the meter box
2. Remove old distribution board and label all existing circuits
3. Mount new PVC flush board in existing cavity (220×210×90mm)
4. Install DIN rail earth leakage unit in slots 1–2
5. Install MCBs in slots 3–8 per circuit layout above
6. Re-terminate existing circuit wiring onto new breakers
7. Verify earth continuity and insulation resistance on all circuits
8. Restore mains supply and test each circuit
9. Electrician issues Certificate of Compliance (CoC)

---

## Notes

- No surge protector included — board is fully populated at 8 modules
- All breakers are DIN rail mount (not CBI or mini rail)
- 32A MCB used for geyser because 30A DIN rail is not stocked at Cashbuild
- Earth leakage is double-pole (occupies 2 module slots)
- Existing cable sizes should be verified by electrician before re-termination
- Standard cable sizes for reference: 1.5mm² (lights), 2.5mm² (plugs), 4mm² (geyser), 6mm² (stove)
